# Fresh Fork

Fresh Fork is a React-based UI project that showcases animated food sections including salads, healthy dishes, and smoothies. The focus of this project is on **smooth animations, visual transitions, and interactive UI design**, not on browsing or ordering food.

## Features
- Animated hero section with rotating salad display
- Scroll-based animations for healthy dishes
- Interactive smoothie slider with transitions
- Clean and modern food-themed UI

## Tech Stack
- React.js
- CSS animations
- JavaScript (Hooks: useState, useEffect)
